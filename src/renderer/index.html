元认知级别的 Agent 自演进
开启后，每个 Agent 会在工作一段时间后自动复盘最近的表现，总结经验、调整自己的工作方式，下次接到任务时表现更好。称为「元认知」是因为 Agent 不只在做任务，还会反思「自己做得怎样、下次怎么做更好」——这种对自身认知过程的认知就是元认知。关闭后 Agent 不再自动复盘。
